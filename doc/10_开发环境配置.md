# 开发环境配置

## 10.1 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11、macOS、Linux |
| 内存 | 8GB 及以上 |
| 硬盘空间 | 2GB 可用空间 |
| Node.js | 14.x 及以上（推荐 LTS 版本） |

## 10.2 必需工具

### 10.2.1 LayaAir IDE

LayaAir IDE 是项目开发的核心工具，用于场景编辑、UI 设计和项目构建。

**下载地址**：https://www.layabox.com/

**版本要求**：3.1.3 或兼容版本

**安装步骤**：
1. 访问 LayaAir 官网下载对应版本
2. 运行安装程序，按提示完成安装
3. 启动 IDE，选择工作空间目录

### 10.2.2 微信开发者工具

用于微信小程序的调试和预览。

**下载地址**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**版本要求**：稳定版最新版

**配置要求**：
- 需要微信小程序 AppID（项目已配置：wx47483884ad0a7a56）
- 开启 ES6 转 ES5（项目设置中已启用）

### 10.2.3 代码编辑器（推荐 VS Code）

**下载地址**：https://code.visualstudio.com/

**推荐插件**：

| 插件名 | 用途 |
|--------|------|
| TypeScript Importer | TypeScript 自动导入 |
| ESLint | 代码规范检查 |
| Prettier | 代码格式化 |

## 10.3 项目初始化

### 10.3.1 克隆项目

```bash
git clone https://gitlab.com/JaggerG/economicadvanture.git
cd economicadvanture
```

### 10.3.2 安装依赖

```bash
cd src
npm install
```

依赖安装完成后，`src/node_modules/` 目录下会包含：
- `decimal.js` — 高精度数值计算库
- `crypto-js` — 加密库

### 10.3.3 LayaAir IDE 打开项目

1. 启动 LayaAir IDE
2. 选择 `文件 -> 打开项目`
3. 选择项目根目录 `EconomicAdvanture.laya` 文件
4. 等待项目加载完成

## 10.5 开发工作流

### 10.5.1 场景编辑

1. 在 LayaAir IDE 中打开 `assets/Scene.ls`
2. 使用可视化编辑器调整 UI 布局
3. 绑定脚本组件到对应节点
4. 保存场景文件

### 10.5.2 代码编写

1. 在 VS Code 中打开 `src/` 目录
2. 编辑 TypeScript 脚本文件
3. 利用类型定义文件（`engine/types/`）获得代码提示

### 10.5.3 本地调试

**Web 调试**：
1. 在 LayaAir IDE 中点击 `运行 -> 运行`
2. 自动打开浏览器预览
3. 使用浏览器开发者工具调试

**微信小程序调试**：
1. 在 LayaAir IDE 中构建项目（选择微信小游戏平台）
2. 打开微信开发者工具
3. 导入构建输出的目录
4. 使用微信开发者工具调试

### 10.5.4 构建发布

1. 在 LayaAir IDE 中选择 `发布 -> 发布项目`
2. 选择目标平台（Web/微信小游戏等）
3. 配置构建选项
4. 点击发布，输出到 `bin/` 或指定目录

## 10.6 常见问题

### Q: LayaAir IDE 无法识别项目？

**A:** 确保打开的是 `.laya` 项目文件，而不是文件夹。如果项目文件损坏，可以尝试新建项目并导入资源。

### Q: TypeScript 编译报错？

**A:** 检查以下几点：
- Node.js 版本是否满足要求
- `tsconfig.json` 配置是否正确
- 依赖是否完整安装

### Q: 微信开发者工具预览白屏？

**A:** 可能原因及解决方案：
- 检查构建输出路径是否正确
- 确认 `project.config.json` 中的 `appid` 有效
- 查看控制台报错信息

### Q: 场景中的脚本绑定丢失？

**A:** 确保：
- 脚本文件路径正确（相对 `src/` 目录）
- 脚本类名与文件名一致
- 使用了 `@regClass()` 装饰器
