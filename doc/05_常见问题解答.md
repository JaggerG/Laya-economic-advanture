# 常见问题解答（FAQ）

## 5.1 基础问题

### Q1: 游戏无法加载或卡在启动画面怎么办？

**A:** 请尝试以下步骤：
1. 检查网络连接是否正常
2. 清理微信缓存后重新进入
3. 确保微信版本为最新版
4. 如仍有问题，请稍后再试或联系客服

### Q2: 科学点数有什么作用？

**A:** 科学点数是游戏中的核心货币，主要用于：
- 升级卡牌等级
- 提升资产的生产效率
- 通过任务奖励获取

### Q3: 资产的生产速度如何计算？

**A:** 生产速度受以下因素影响：
- **基础速度**：由资产的 `time` 属性决定
- **Speed 加成**：卡牌提供的速度倍率
- **实际周期**：`基础周期 / Speed加成`

例如：基础周期4秒，Speed加成2倍，则实际周期为2秒。

### Q4: 为什么有时候购买按钮显示不能购买？

**A:** 购买资产需要消耗对应资源，当资源不足时：
- Parent 资源（对应资源类型）不足
- Employee 资源不足
- 上级资产数量不足（购买高级资产时）

请检查资源数量或切换购买批量模式。

## 5.2 卡牌系统问题

### Q5: 如何获得新卡牌？

**A:** 卡牌主要通过以下途径获得：
1. **任务奖励**：完成任务后随机抽取
2. **初始赠送**：部分卡牌在游戏开始时已解锁

### Q6: 卡牌升级需要什么条件？

**A:** 卡牌升级需要满足两个条件：
1. **收集数量**：已收集的卡牌数量达到升级目标
2. **科学点数**：拥有足够的科学点数支付升级费用

### Q7: 不同卡牌的加成效果可以叠加吗？

**A:** 可以。同一资产的多张卡牌加成按乘法计算：
- 最终效果 = 基础值 × 卡牌A倍率 × 卡牌B倍率 × ...

### Q8: 为什么有些卡牌显示 "locked"？

**A:** 显示 "locked" 表示该卡牌尚未解锁。解锁条件通常为：
- 首次获得该卡牌（通过任务奖励抽取）
- 达到指定等级要求

## 5.3 任务系统问题

### Q9: 任务进度为什么不增加？

**A:** 请确认：
1. 任务类型对应的操作是否正确
2. 对于 COLLECT 类型，需要主动收集才会计数
3. 对于 SPEND 类型，只有花费科学点数才计数

### Q10: 任务完成后如何领取奖励？

**A:** 当任务进度条满时：
1. 任务右侧会出现完成标记
2. 点击完成标记即可领取奖励
3. 系统会自动抽取卡牌和科学点数

### Q11: 任务会刷新吗？

**A:** 会。完成任务后：
- 当前任务槽位会自动分配新任务
- 新任务从任务配置池中随机选取
- 当所有任务都完成后，会循环或进入下一等级

## 5.4 数值与显示问题

### Q12: 数值后面的 K、M、B 是什么意思？

**A:** 这是大数值的缩写显示：

| 缩写 | 数值 |
|------|------|
| K | 千（1,000） |
| M | 百万（1,000,000） |
| B | 十亿（1,000,000,000） |
| T | 万亿（1,000,000,000,000） |
| Q | 千万亿 |
| AA-ZZ | 更大数值 |

### Q13: 为什么购买数量显示为小数？

**A:** 在 10% 和 50% 批量模式下，系统会计算最大可购数量的百分比。如果计算结果小于1，系统会自动调整为1，确保至少能购买1个。

### Q14: 暴击是如何触发的？

**A:** 暴击触发机制：
1. 每次生产时，系统生成 0-100 的随机数
2. 如果随机数 ≤ Chance 加成值，则触发暴击
3. 暴击时，产出数量 × Bonus 倍率

## 5.5 操作与界面问题

### Q15: 如何切换不同的资源面板？

**A:** 在主场景中：
1. 查看顶部的资源标签栏
2. 点击想要查看的资源名称（如 Potato、Orange）
3. 下方资产面板会切换到对应资源

### Q16: 底部导航的各页面有什么功能？

**A:** 底部导航当前功能：

| 按钮 | 功能 |
|------|------|
| 0 | 主场景（资源生产） |
| 1 | 预留功能页（暂未开放） |
| 2 | 卡牌系统 |
| 3-4 | 预留扩展页（暂未开放） |

### Q17: 游戏支持离线收益吗？

**A:** 当前版本暂未实现离线收益功能。游戏需要保持运行状态才能获得资产产出。后续版本可能会增加此功能。

## 5.6 技术问题

### Q18: 游戏支持哪些平台？

**A:** 当前版本主要支持：
- 微信小程序（主要平台）
- 兼容移动端浏览器

### Q19: 游戏数据会保存吗？

**A:** 当前版本数据保存在本地。建议：
- 不要清理微信缓存，否则数据可能丢失
- 后续版本会增加云端存档功能

### Q20: 遇到 Bug 如何反馈？

**A:** 请通过以下方式反馈：
1. 记录问题发生的具体步骤
2. 截图保存问题界面
3. 联系开发团队或提交 Issue
