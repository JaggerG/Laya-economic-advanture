## 1. 项目概述

### 1.1 项目基本信息

| 项目属性 | 内容 |
|---------|------|
| 项目名称 | Economic Adventure（经济冒险） |
| 游戏类型 | 移动端放置类经济模拟小游戏 |
| 游戏引擎 | LayaAir 3.1.3 |
| 开发语言 | TypeScript (ES6) |
| 目标平台 | 微信小程序（主要）、Web 浏览器（兼容） |
| 设计分辨率 | 399 x 844（竖屏） |
| 当前版本 | v1.0.0 |

### 1.2 技术架构

```
+---------------------+
|     表现层（UI）      |
|  - Scene 场景管理     |
|  - UI 组件渲染        |
|  - 动画与特效         |
+---------------------+
|     逻辑层（Logic）   |
|  - 游戏主逻辑         |
|  - 资产生产计算       |
|  - 任务状态管理       |
|  - 卡牌系统逻辑       |
+---------------------+
|     数据层（Data）    |
|  - 配置数据 (model/)  |
|  - 玩家存档 (localStorage) |
|  - 运行时状态         |
+---------------------+
|     引擎层（Engine）  |
|  - LayaAir 3.1.3     |
|  - 渲染引擎 / 事件系统 / 定时器 |
+---------------------+
|     平台层（Platform）|
|  - 微信小程序 / Web   |
+---------------------+
```

### 1.3 核心依赖

| 依赖库 | 版本 | 用途 |
|--------|------|------|
| decimal.js | ^7.4.0 | 高精度数值计算 |
| crypto-js | ^4.2.0 | 加密算法（预留） |
| @types/crypto-js | ^4.2.2 | 类型定义 |

**等级提升流程：**
```
完成任务 → 获得 EXP → 累加到 userInfo.exp
    ↓
检查 userInfo.exp >= Player_Level_Config[userInfo.level + 1].exp_required
    ↓
是 → level++，触发 LevelUp 事件 → UI 播放升级特效
    ↓
检查是否有新卡牌解锁 → 触发 UnlockCard 事件
```

**已实现的核心逻辑（`src/utils/LevelManager.ts`）：**
- `addExp(amount)`：增加经验值，自动检测升级，支持连续多级升级
- `getLevelTitle(level)`：获取等级称号
- `getExpProgress()`：获取当前 EXP / 升级所需 EXP
- `getUnlockedCards()`：获取当前等级已解锁的卡牌列表
- 派发事件：`Exp_update`、`LevelUp`、`UnlockCard`
