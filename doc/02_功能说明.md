# 功能说明

## 2.1 主场景功能

### 2.1.1 资源生产面板

主场景包含多个资源标签页（ViewStack），每个标签页对应一种资源类型：

| 资源类型 | 说明 |
|---------|------|
| Potato（土豆） | 基础资源，初始可用 |
| Orange（橙子） | 进阶资源，需要解锁 |
| Tomato（番茄） | 高级资源，需要解锁 |

每个资源面板包含多个资产项（AssetItem），资产会自动生产对应资源。

### 2.1.2 资产系统

每个资产包含以下属性：

| 属性 | 说明 |
|------|------|
| name | 资产名称（如 Farmer、Commune） |
| amount | 当前拥有数量 |
| level | 当前等级 |
| outcome | 每次产出数量 |
| produce | 产出目标（Parent 或上级资产） |
| time | 生产周期（秒） |
| cost | 购买消耗 |
| bonus | 五维加成属性 |

### 2.1.3 五维加成系统

每个资产拥有五种加成属性：

| 加成类型 | 作用 |
|---------|------|
| Power（力量） | 提升产出数量 |
| Discount（折扣） | 降低购买消耗 |
| Speed（速度） | 缩短生产周期 |
| Chance（暴击率） | 触发暴击产出的概率 |
| Bonus（暴击倍率） | 暴击时的额外倍率 |

## 2.2 卡牌系统

### 2.2.1 卡牌类型

卡牌按加成类型分为五类，对应资产的五维加成：

- Power 卡牌
- Discount 卡牌
- Speed 卡牌
- Chance 卡牌
- Bonus 卡牌

### 2.2.2 卡牌属性

| 属性 | 说明 |
|------|------|
| name | 卡牌名称 |
| type | 加成类型 |
| related_assets | 关联资产 |
| level_ratio | 每级加成倍率 |
| level | 当前等级 |
| has_amount | 已收集数量 |
| cost | 升级消耗的科学点数 |
| weight | 抽卡权重 |

### 2.2.3 卡牌升级

- 收集足够数量的同名卡牌可升级
- 升级消耗科学点数
- 升级后加成效果按 `level_ratio^level` 计算

## 2.3 任务系统

### 2.3.1 任务类型

| 类型 | 说明 | 示例 |
|------|------|------|
| OWN | 拥有指定数量的资产 | 拥有250个农民 |
| SPEND | 花费指定数量的科学点数 | 花费25科学点 |
| COLLECT | 收集指定数量的资源/卡牌 | 收集250个土豆 |

### 2.3.2 任务奖励

完成任务可获得：

- **卡牌奖励**：随机抽取 `c_min` ~ `c_max` 张卡牌，保底 `atLeast` 张
- **科学点数**：随机获得 `s_min` ~ `s_max` 点科学点数

## 2.4 雇员系统

雇员（Employee）是独立的生产单位：

- 自动生产雇员点数
- 用于购买资产的消耗材料之一
- 生产速度和产量可通过配置调整

## 2.5 底部导航

底部导航栏包含5个标签页，用于切换主功能场景：

| 索引 | 功能 |
|------|------|
| 0 | 主场景（资源生产） |
| 1 | 预留功能页 |
| 2 | 卡牌系统 |
| 3-4 | 预留扩展页 |

## 2.6 购买批量选择

支持四种购买模式：

| 模式 | 说明 |
|------|------|
| x1 | 购买1个 |
| 10% | 购买最大可购数量的10% |
| 50% | 购买最大可购数量的50% |
| 最大 | 购买最大可购数量 |
