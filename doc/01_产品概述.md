# 产品概述

## 1.1 产品名称

**Economic Adventure（经济冒险）**

## 1.2 产品定位

Economic Adventure 是一款基于 LayaAir 引擎开发的移动端放置类经济模拟小游戏。玩家通过收集资源、升级资产、收集卡牌和完成任务，逐步建立和扩展自己的经济帝国。游戏采用竖屏设计，适配微信小程序平台，具有轻量级、易上手的特点。

## 1.3 核心玩法

游戏的核心循环围绕以下机制展开：

- **资源生产**：通过购买和升级各类资产（如农民、公社等），自动生产土豆、橙子、番茄等资源
- **卡牌系统**：收集和升级各类加成卡牌，提升资源生产效率
- **任务系统**：完成各类任务目标，获取卡牌和科学点数奖励
- **科学点数**：用于卡牌升级的核心货币，通过任务奖励获得

## 1.4 目标用户

- 喜欢放置类和模拟经营类游戏的休闲玩家
- 追求数值成长和策略优化的核心玩家
- 利用碎片时间进行游戏的移动端用户

## 1.5 产品特色

| 特色 | 说明 |
|------|------|
| 放置玩法 | 资产自动生产，离线也有收益 |
| 卡牌养成 | 多类型卡牌收集与升级系统 |
| 任务驱动 | 丰富的任务系统引导玩家成长 |
| 数值膨胀 | 支持超大数值显示（K/M/B/T 等单位） |
| 竖屏适配 | 专为移动端竖屏操作优化 |

## 1.6 版本信息

- **当前版本**：v1.0.0
- **引擎版本**：LayaAir 3.1.3
- **微信小程序**：v3.5.0
- **设计分辨率**：399 x 844（竖屏）
