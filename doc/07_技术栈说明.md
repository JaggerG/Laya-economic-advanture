# 技术栈说明

## 7.1 核心引擎

### LayaAir Engine 3.1.3

LayaAir 是一款国产的高性能 HTML5 游戏引擎，支持 2D/3D 游戏开发，本项目使用其 2D UI 模块。

| 特性 | 说明 |
|------|------|
| 渲染模式 | Canvas / WebGL 双模式 |
| 脚本语言 | TypeScript |
| UI 系统 | 内置 UI 组件库（ViewStack、ProgressBar、Label 等） |
| 场景编辑 | 可视化场景编辑器（LayaAir IDE） |
| 构建输出 | HTML5 / 微信小程序 / 其他小游戏平台 |

### 引擎模块启用

```json
{
  "modules": {
    "laya.d3": false,    // 3D 模块：禁用
    "laya.ui": true      // UI 模块：启用
  }
}
```

## 7.2 开发语言

### TypeScript

- **版本目标**：ES6
- **模块系统**：ES6 Module
- **严格模式**：启用（strictNullChecks 关闭）
- **装饰器**：启用 experimentalDecorators

### tsconfig.json 配置

```json
{
  "compilerOptions": {
    "module": "es6",
    "target": "es6",
    "strict": true,
    "strictNullChecks": false,
    "noEmitHelpers": true,
    "sourceMap": false,
    "experimentalDecorators": true,
    "skipLibCheck": true,
    "moduleResolution": "node",
    "allowSyntheticDefaultImports": true
  },
  "include": ["./assets", "./src", "./engine"]
}
```

## 7.7 代码规范

### 装饰器模式

使用 LayaAir 提供的装饰器注册脚本组件：

```typescript
const { regClass, property } = Laya;

@regClass()
export class Main extends Laya.Script {
  @property({ type: Laya.ViewStack })
  public viewStack: Laya.ViewStack;
}
```

### 事件命名规范

| 事件类型 | 命名格式 | 示例 |
|---------|---------|------|
| 数据更新 | `{name}_update` | `Science_update` |
| 收集更新 | `{name}_COLLECT_update` | `Potato_COLLECT_update` |
| 花费更新 | `{name}_SPEND_update` | `Science_SPEND_update` |
| 任务进度 | `task_progress_update` | - |
