# 使用场景

## 4.1 场景一：新手入门体验

### 场景描述
玩家首次进入游戏，需要快速理解核心玩法并建立初始经济循环。

### 操作流程

1. **进入主场景**
   - 加载完成后自动进入 Potato 资源面板
   - 顶部显示雇员进度和科学点数（初始10点）

2. **购买首个资产**
   - 点击 Farmer 的 "购买 x1" 按钮
   - 消耗 Parent 资源和 Employee 资源
   - Farmer 数量增加，开始自动生产土豆

3. **观察生产**
   - 进度条自动填充，每4秒产出一次
   - 土豆数量自动增加

4. **完成任务**
   - 当拥有250个 Farmer 时，第一个任务完成
   - 点击完成按钮，获得卡牌和科学点数奖励

### 关键体验点
- 即时反馈：购买后立即看到资产增加
- 自动化：购买后无需操作即可持续收益
- 目标引导：任务系统提供明确的短期目标

## 4.2 场景二：卡牌收集与升级

### 场景描述
玩家通过任务奖励获得卡牌，并决定如何分配科学点数升级卡牌。

### 操作流程

1. **获得卡牌奖励**
   - 完成任务后，系统随机抽取卡牌
   - 卡牌自动加入玩家的卡牌收藏

2. **查看卡牌**
   - 点击底部导航 "2" 进入卡牌页面
   - 查看已解锁的卡牌和收集进度

3. **选择升级目标**
   - 点击目标卡牌，查看详情
   - 对比不同卡牌的加成效果

4. **执行升级**
   - 收集进度满后，点击升级按钮
   - 消耗科学点数，卡牌等级提升
   - 对应资产的加成效果立即生效

### 决策要点
| 卡牌类型 | 升级优先级 | 适用阶段 |
|---------|-----------|---------|
| Speed | 高 | 全阶段 |
| Power | 高 | 中后期 |
| Discount | 中 | 资源紧张时 |
| Chance | 中 | 暴击体系成型后 |
| Bonus | 低 | 高 Chance 后 |

## 4.3 场景三：批量购买与资源规划

### 场景描述
玩家拥有大量资源后，需要高效购买资产并规划资源使用。

### 操作流程

1. **切换批量模式**
   - 点击批量按钮，切换至 "50%" 或 "最大"

2. **批量购买**
   - 连续点击多个资产的购买按钮
   - 快速扩充资产规模

3. **资源监控**
   - 观察 Parent 资源和 Employee 资源的消耗
   - 确保不会因过度购买导致生产停滞

4. **切换资源类型**
   - 当一种资源充足时，切换到其他资源面板
   - 均衡发展多种资源线

## 4.4 场景四：任务链推进

### 场景描述
玩家通过连续完成任务，推进游戏进度并获取大量奖励。

### 操作流程

1. **查看当前任务**
   - 顶部任务栏显示3个并行任务
   - 每个任务有不同的完成条件

2. **针对性操作**
   - 根据任务类型，选择对应操作：
     - OWN 任务：集中购买目标资产
     - SPEND 任务：主动升级卡牌消耗科学点
     - COLLECT 任务：等待资产自动生产或主动收集

3. **领取奖励**
   - 任务完成后点击领取
   - 观看（或跳过）抽卡动画
   - 获得卡牌和科学点数

4. **任务刷新**
   - 系统自动分配新任务
   - 整体任务进度条推进

## 4.5 场景五：多资源线管理

### 场景描述
游戏后期，玩家需要同时管理 Potato、Orange、Tomato 三条资源线。

### 操作流程

1. **资源标签切换**
   - 点击顶部标签栏切换不同资源
   - 每条资源线有独立的资产体系

2. **优先级判断**
   - 根据当前任务需求，决定优先发展哪条资源线
   - 根据卡牌加成情况，选择效率最高的资源线

3. **跨资源协同**
   - 某些高级资产的购买需要其他资源的产出
   - 合理分配资源，避免单一资源过度消耗

## 4.6 典型玩家旅程

```
第1-5分钟：熟悉界面，购买首批资产，完成首个任务
第5-30分钟：建立自动生产循环，收集首批卡牌，升级1-2张卡牌
第30分钟-2小时：多资源线并行发展，卡牌组合策略形成
第2小时+：追求极致效率，完成高级任务，解锁全部卡牌
```
