# 用户指南

## 3.1 快速开始

### 3.1.1 进入游戏

1. 打开微信小程序，搜索并进入 "Economic Adventure"
2. 等待游戏加载完成，进入主场景
3. 初始会获得10点科学点数

### 3.1.2 界面布局

游戏主界面采用竖屏布局，从上到下依次为：

```
+------------------+
|   顶部信息栏      |  <- 雇员进度、科学点数、任务进度
+------------------+
|   资源标签栏      |  <- 切换不同资源类型
+------------------+
|                  |
|   资产生产面板    |  <- 显示资产列表和生产进度
|                  |
+------------------+
|   底部导航栏      |  <- 切换主功能页面
+------------------+
```

## 3.2 基础操作

### 3.2.1 购买资产

1. 在资产生产面板中，找到想要购买的资产
2. 点击资产右侧的 **"购买"** 按钮
3. 系统会自动扣除对应资源，增加资产数量

### 3.2.2 切换资源类型

1. 点击顶部的资源标签栏（如 Potato、Orange、Tomato）
2. 面板会切换到对应资源的生产页面
3. 不同资源的资产相互独立

### 3.2.3 切换购买数量

1. 点击资源标签栏右侧的批量按钮（默认显示 "x1"）
2. 按钮会在 x1 / 10% / 50% / 最大 之间循环切换
3. 切换后所有资产的购买按钮会同步更新

## 3.3 卡牌系统操作

### 3.3.1 查看卡牌

1. 点击底部导航栏的 **"2"** 标签
2. 进入卡牌页面，查看所有已解锁和未解锁的卡牌

### 3.3.2 卡牌详情

1. 点击任意卡牌，弹出卡牌详情对话框
2. 对话框显示：
   - 卡牌名称和描述
   - 当前收集进度
   - 升级所需科学点数

### 3.3.3 升级卡牌

1. 在卡牌详情页，当收集进度满时，**升级按钮**会显示
2. 点击升级按钮，消耗对应科学点数
3. 卡牌等级提升，加成效果增强

## 3.4 任务系统操作

### 3.4.1 查看任务

顶部信息栏下方显示当前任务列表，每个任务包含：
- 任务描述
- 进度条
- 完成状态

### 3.4.2 完成任务

1. 任务进度条满后，任务右侧会出现完成按钮
2. 点击完成按钮，领取奖励
3. 系统会自动抽取卡牌和科学点数奖励
4. 完成后会自动刷新为新任务

### 3.4.3 任务进度计算

| 任务类型 | 进度计算方式 |
|---------|------------|
| OWN | 当前拥有资产数量 / 目标数量 |
| SPEND | 已花费科学点数 / 目标数量 |
| COLLECT | 已收集资源数量 / 目标数量 |

## 3.5 数值说明

### 3.5.1 数值单位

当数值较大时，游戏会自动转换为缩写单位：

| 单位 | 数值 |
|------|------|
| K | 1,000 |
| M | 1,000,000 |
| B | 1,000,000,000 |
| T | 1,000,000,000,000 |
| Q | 1,000,000,000,000,000 |
| AA | 10^15 |
| BB | 10^18 |
| ... | ... |
| ZZ | 10^81 |

### 3.5.2 生产效率计算

资产的每秒产出 = `资产数量 × 单次产出 × Power加成 / (生产周期 / Speed加成)`

## 3.6 游戏技巧

1. **优先升级 Speed 卡牌**：缩短生产周期，提升整体效率
2. **平衡购买和升级**：不要只买资产不升级卡牌
3. **及时完成任务**：任务是获取卡牌和科学点数的主要来源
4. **关注暴击率**：Chance 和 Bonus 卡牌可带来额外收益
